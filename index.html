Core Features

The application is built for speed, offline reliability, and deep Android customization.

Unified Week Layout

Features perfect Saturday grid mapping with clean overflow layouts, ensuring weekend lectures remain visible and aligned.

Wallpaper-Adaptive Widgets

Scans system wallpaper color palettes programmatically to swap widget text colors (deep dark vs neon white) maintaining readability.

Study Mode Auto-Mute

Silences incoming call and message alerts automatically during scheduled lecture intervals using native System services.

Attendance Analytics

Provides visual rings showing target present ratios, letting you monitor necessary class attendance criteria.

Offline SQLite Engine

Saves all courses, classes, and tasks locally to your storage for lightning-fast retrievals without requiring internet access.

System Calendar Exporter

Instantly exports your timetable to Google Calendar or system calendar databases with one tap from the settings panel.
